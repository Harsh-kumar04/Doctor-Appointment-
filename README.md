# 🩺 Doctor Appointment

A **modern and user-friendly web application** for seamless **doctor appointment booking**.  
Crafted with care using **Figma**, the interface ensures a smooth experience for both patients and doctors.

---

## 🚀 Features

### 👤 User Authentication
- Secure **registration**, **login**, and **logout** for both patients and doctors.  
- Role-based access for privacy and data protection.


### 📅 Appointment Booking
- Easy-to-use forms to **book, view, and manage appointments**.  
- Real-time updates for accurate scheduling.

### 🧑‍💼 Dashboard
**Patients:**  
- View **upcoming appointments** and **appointment history**.  
- Manage **personal profiles** easily.  

**Doctors:**  
- Manage **available slots** and **booked appointments**.  
- Maintain a **patient list** for follow-ups.  






## 💻 Tech Stack

| Category | Technology |
|-----------|-------------|

| **Backend** | Node.js, Express *(RESTful APIs)* |
| **Database** |   MySQL  |
| **Authentication** | JWT  |
| **Other Tools** | Nodemailer *(email notifications)*, dotenv *(environment variables)* |
